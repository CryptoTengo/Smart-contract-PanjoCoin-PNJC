# 🪙 PanjoCoin (PNJC) — Tokenomics

## Overview

PanjoCoin (PNJC) is a fixed-supply ERC-20 token deployed on Polygon. All allocations are defined at genesis and fully verifiable on-chain.

No minting or inflation mechanisms exist.

---

## Core Parameters

- Network: Polygon
- Standard: ERC-20
- Total Supply: 1,000,000,000,000 PNJC
- Decimals: 18
- Minting: Disabled

---

## Allocation Model

See README for official allocation table.

All tokens are assigned to dedicated wallets:

- Liquidity Wallet
- Treasury Wallet
- DAO Wallet
- Team Wallet
- Founder Wallet
- Community Wallet
- Charity Wallet

---

## Enforcement Model

All allocations are enforced through:

- Genesis wallet distribution
- Smart contract vesting (for locked allocations)
- Multisig governance (for treasury control)

No off-chain accounting exists.

---

## Supply Behavior

- Fixed supply
- No inflation
- No rebasing
- No hidden mint functions
