# PanjoCoin Smart Contract (PNJC)

This repository contains the official smart contract for **PanjoCoin (PNJC)** — a utility meme token deployed on the Polygon blockchain.

## Overview

PanjoCoin (PNJC) is an ERC-20 token designed to combine community engagement with real-world impact.
A portion of the project is dedicated to supporting **clown therapy initiatives** at the M. Iashvili Children's Central Hospital in Tbilisi, specifically in the Oncology-Hematology and Neurosurgery departments.

## Token Details

* **Token Name:** PanjoCoin
* **Symbol:** PNJC
* **Standard:** ERC-20
* **Blockchain:** Polygon
* **Total Supply:** 1,000,000,000,000 PNJC
* **Decimals:** 18

## Smart Contract

The smart contract is written in Solidity and follows the ERC-20 standard using OpenZeppelin libraries for security and reliability.

### Features

* Standard ERC-20 functionality
* Fixed total supply
* Secure implementation based on OpenZeppelin
