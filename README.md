# PanjoCoin Smart Contract (PNJC)

This repository contains the official smart contract for **PanjoCoin (PNJC)** — a decentralized utility meme token deployed on the **Polygon** blockchain.

## Overview

PanjoCoin (PNJC) is an ERC-20 token designed to combine community engagement with real-world impact. A significant portion of the project is dedicated to supporting **medical clowning (ClownCare) initiatives** in Tbilisi, Georgia. Specifically, the project aims to fund professional clown therapy at the **M. Iashvili Children's Central Hospital**, focusing on the Oncology-Hematology and Neurosurgery departments.

## Token Details

* **Token Name:** PanjoCoin
* **Symbol:** PNJC
* **Standard:** ERC-20 (with Burnable & Permit extensions)
* **Blockchain:** Polygon (Mainnet)
* **Total Supply:** 1,000,000,000,000 PNJC
* **Decimals:** 18

## Smart Contract

The contract is written in Solidity 0.8.25 and utilizes **OpenZeppelin v5.0.2** libraries to ensure industry-standard security and reliability.

**Official Contract Address (Polygon):** `0xdA25Ed82Cf814f0a30f6C95231c3B77056E18286`

## Features

* **Fixed Supply:** No additional tokens can be minted, ensuring long-term scarcity.
* **Burnable:** Users can burn their tokens to reduce the circulating supply.
* **EIP-2612 Permit:** Supports gasless approvals for better user experience.
* **Zero-Admin:** The contract is fully decentralized with no owner/admin functions.

## Deployment

The contract was successfully compiled using the **Cancun EVM** and deployed via **Remix IDE** with 200 optimization runs.

## Security

* **OpenZeppelin Standards:** Built using battle-tested contract templates.
* **Verified Source Code:** The contract is verified and public on Polygonscan.
* **Independent Audit:** While built on secure standards, users are encouraged to perform their own due diligence.

## Purpose & Goals

PanjoCoin aims to:
* Build a strong, community-driven ecosystem.
* Provide utility through future integrations (DEX, rewards, and charity-linked NFTs).
* Scale **ClownCare** projects from Tbilisi to regional and international hospitals.
