# 🪙 PanjoCoin (PNJC) — Tokenomics

Overview

PanjoCoin (PNJC) is a fixed-supply ERC-20 token deployed on the Polygon network.
All token allocations are defined at genesis and are fully verifiable on-chain.

There is no minting, inflation, or hidden supply expansion.

Core Parameters
Network: Polygon
Standard: ERC-20
Total Supply: 1,000,000,000,000 PNJC
Decimals: 18
Minting: Disabled
Allocation Model

All PNJC tokens are distributed at genesis into dedicated wallets, each serving a specific role in ecosystem governance, liquidity, operations, and community growth.
