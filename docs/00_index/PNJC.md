# 1. Protocol Overview

PanjoCoin (PNJC) is a modular ERC-20 based digital asset protocol deployed on the Polygon blockchain.

The protocol is designed as a non-upgradeable, fixed-supply token ecosystem with modular governance, treasury management, staking, vesting, and liquidity coordination layers.

The architecture prioritizes:

- supply immutability
- modular security isolation
- transparent treasury operations
- decentralized governance readiness
- minimized smart contract attack surface
